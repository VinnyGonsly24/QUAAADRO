# QUAAADRO Web

Versao PWA para GitHub Pages.

Arquivos principais:

- `index.html`: aplicativo web
- `manifest.webmanifest`: instalacao como app
- `sw.js`: cache basico
- `icons/`: icones do app

Ao abrir em outro computador, entre com a mesma conta de e-mail e senha criada no app para baixar os projetos da nuvem.
